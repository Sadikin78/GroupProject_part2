    Enhancements

    Flexbox

    Flexbox is a way to layout elements in rows or columns that extends into available space or fills an area on a page.


    This enhancement overcomes the irritating limitations of arranging items on a page, such as via floating, and enables greater flexibility. Flexbox is not taught in the Computing Technology Inquiry Project unit and reaches beyond the basic requirements because of the numerous behaviours applied with this method.


    Implementing flexbox in CSS requires display: flex; to be included in the parent container. Child elements can be arranged as flex-direction: row; or flex-direction: column; by specifying this in the container element. A useful option is to centre elements which may be achieved in a few ways. justify-content: center; can be used to align the content horizontally in the container and align-items: center; allows vertical centring in the container. Other flexbox options exist including changing the order of items and wrapping content.


    Flexbox has been implemented with guidance from Flexbox Sheet (Joni Trythall).

    Responsive Design

    Media queries are used to change elements on the page based on the user’s device type or from certain characteristics, namely the viewport width.


    Including media queries as an enhancement allows the site to be used with different viewports effectively regardless of whether the website is viewed on a computer, tablet, or mobile device browser. This technique has not been taught in the unit and expands the basic requirements of the project because items on the site are rearranged and modified to tailor the user’s device, which in turn ensures legibility is maintained.


    To implement responsive web design, the viewport is set for all HTML pages so the browser can fit content appropriately on the user’s device, by using: <meta name="viewport" content="width=device-width, initial-scale=1.0">. A media query is then used in CSS to define changes to elements when the browser window is lower than a specified measurement. For instance, @media only screen and (max-width: 900px) specifies that style changes will occur when the browser’s width is lower than 900px. Element style changes are specified inside the media query.


    Media queries have been implemented with guidance from Responsive Web Design Media Queries (w3schools.com).
